# Introduction to Finko UI

Finko UI was designed to be modern and to remind the lovely Microsoft Money navigation style using colorful spaces.  
We call this UI **Nostalgic UI**. You switch between **Nostalgic Light**, **Nostalgic Dark**, **Modern Light** and **Modern Dark** themes.  
The two "Modern" themes won't provide colorful spaces and will only use the application accent color which is green for Finko.
