# Display settings

Use the **Display settings** button to adjust the settings for the display in the designer.

**Better display** : Improve the visual appearance of the designer and shapes.  
SVG drawings are smoother, especially in high resolution. Scrolling and zooming are smoother.

**Better performance** : Optimize the performance of the designer and shapes.  
FPS are lowered to save CPU and RAM. Scrolling and zooming are deferred to preserve performance.
