# Vista version 1.0.7.0

Published on 05/24/2026.

## About updates

Updates generally include bug fixes, security fixes, and new features.
We describe the changes in this release below.  
For more information about security and how we protect your data, please read
our [commitment to quality](../../../trust/index.md).

## Compatible platforms

This release was tested on the following platforms:

* Linux Mint 22.3 - Cinnamon 64-bit
* Windows 11 26H1 - 64-bit

## Bug Fixes

* Fixed a bug that caused some format features to be disabled for connections.
* Fixed a bug that caused opacity not to be applied to shapes when exporting to SVG.
* Fixed a crash that could occur when double clicking on a connection point of a shape.
* Fixed a bug that caused markers to be added to the wrong layer.

## Changes

* You can now use the Query language to search shapes.
* Pasting will now insert shapes at the center of the viewport instead of next to the original shape.
* You can now double-click on a marker in the **markers panel** to navigate to it in the designer.
* You can now easily hide/restore all side panels at once (see the **View** menu).
* Added several new shapes.
* Added several new color themes for diagrams.
* The document theme is now saved and persisted in the file.
* The Copy format command has been improved to show a green rectangle when the selection is valid.
* You can now transform several shapes using a small yellow diamond when selected.
* You can now drag move the diagram using the mouse middle button.

# [Linux](#tab/linux)

## Recommended method

We provide a Debian package for Linux, distributed through
the [Vista APT repository](https://sebbouez.github.io/Vista-APT/).  
You can install Vista using the following commands:

```bash
# Download the public GPG key
curl -fsSL https://sebbouez.github.io/Vista-APT/vista.asc | sudo gpg --dearmor -o /etc/apt/keyrings/vista.gpg

# Add the repository with the GPG key verification
echo "deb [signed-by=/etc/apt/keyrings/vista.gpg] https://sebbouez.github.io/Vista-APT/ stable main" | sudo tee /etc/apt/sources.list.d/vista.list

sudo apt update && sudo apt install vista
```
