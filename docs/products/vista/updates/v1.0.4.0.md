# Vista version 1.0.4.0

Tested on :

* Linux Mint 22.3 - Cinnamon 64-bit
* Windows 11 26H1 - 64-bit

## Bug Fixes

* Fixed a bug that made complicated to select and move transparent shapes.
* Fixed a bug that caused the **shape format** panel to display wrong colors after selecting a shape.
* Fixed a visual bug that caused rounded rectangles to look weird with smaller zoom levels.
* Fixed a bug that caused the **opacity** setting to be forgotten when using the **copy format** command.
* Fixed a bug that caused some settings not to be saved.

## Changes

* Text in shapes now appears with full opacity even when you change the opacity of the shape.
* Vista will start a new instance when you open a diagram while you are already editing one.
* Added the **Shape advanced options** command to set special behaviors for shapes.
* Added the possibility to change font size in shapes.
* You can now close the **search** dialog by pressing the **Escape** key.
* The **layers** panel has been reworked.
* Improved the **search** feature by allowing to select a search scope.

# [Linux](#tab/linux)

You can install Vista on Linux using the following commands:

```bash
# Download the public GPG key
curl -fsSL https://sebbouez.github.io/Vista-APT/vista.asc | sudo gpg --dearmor -o /etc/apt/keyrings/vista.gpg

# Add the repository with the GPG key verification
echo "deb [signed-by=/etc/apt/keyrings/vista.gpg] https://sebbouez.github.io/Vista-APT/ stable main" | sudo tee /etc/apt/sources.list.d/vista.list

sudo apt update && sudo apt install vista
```
