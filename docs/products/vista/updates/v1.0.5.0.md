# Vista version 1.0.5.0

Published on 2026 may 2nd.  

## About updates

Updates generally include bug fixes, security fixes, and new features.
We describe the changes in this release below.  
For more information about security and how we protect your data, please read our [commitment to quality](../../../trust/index.md).

## Compatible platforms

This release was tested on the following platforms:

* Linux Mint 22.3 - Cinnamon 64-bit
* Windows 11 26H1 - 64-bit

## Bug Fixes

* The side panels will remain selected after restarting the application.
* Fixed a bug that caused the "hide connection points" setting on shapes not persists.
* Fixed a bug that caused the highlight operation to create an undoable action.

## Changes

* Improved various UI elements and added consistent cursors in designer.
* Added the **Notes** feature.
* Added the [Annotations](../annotations.md) feature.
* Added the [Spotlight](../spotlight.md) feature.
* Pressing the **Escape** key in the formula bar will reset the formula.
* You can now export shapes data to a CSV file.
* You can now export shapes data to a XLSX file.
* You can now export markers to a CSV file.
* You can now export markers to a XLSX file.

# [Linux](#tab/linux)

We provide a Debian package for Linux, distributed through the [Vista APT repository](https://sebbouez.github.io/Vista-APT/).  
You can install Vista using the following commands:

```bash
# Download the public GPG key
curl -fsSL https://sebbouez.github.io/Vista-APT/vista.asc | sudo gpg --dearmor -o /etc/apt/keyrings/vista.gpg

# Add the repository with the GPG key verification
echo "deb [signed-by=/etc/apt/keyrings/vista.gpg] https://sebbouez.github.io/Vista-APT/ stable main" | sudo tee /etc/apt/sources.list.d/vista.list

sudo apt update && sudo apt install vista
```
