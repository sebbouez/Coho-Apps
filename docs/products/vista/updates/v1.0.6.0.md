# Vista version 1.0.6.0

Published on 05/15/2026.

## About updates

Updates generally include bug fixes, security fixes, and new features.
We describe the changes in this release below.  
For more information about security and how we protect your data, please read
our [commitment to quality](../../../trust/index.md).

## Compatible platforms

This release was tested on the following platforms:

* Linux Mint 22.3 - Cinnamon 64-bit
* Windows 11 26H1 - 64-bit

> [!IMPORTANT]
> This update is not available on Linux.

## Bug Fixes

* Fixed a bug that caused shapes background to become transparent in a specific scenario.
* Fixed a regression that caused some format settings to be disabled when selecting a svg shape.
* Changing the zoom level now stays centered on the current view.
* Fixed several translations.
* Fixed a bug that caused connections to remain visible after undoing a create action.

## Changes

* Reworked the process of loading and saving files.
* Reworked the **Shapes panel**.
* Copy/Paste of markers will now automatically increment them (you may need to start with new markers groups).
* You can now undo/redo actions in the **Shape data** panel.
* You can now undo/redo changes applied to shape annotations.
* You can now export your diagram as a SVG file.
* Changed the PDF export to use vector graphics.
* You can now export all your diagram pages in the **export dialog**.
* You can now set the default arrow end head type in **application settings**.

# [Linux](#tab/linux)

## Recommended method

We provide a Debian package for Linux, distributed through
the [Vista APT repository](https://sebbouez.github.io/Vista-APT/).  
You can install Vista using the following commands:

```bash
# Download the public GPG key
curl -fsSL https://sebbouez.github.io/Vista-APT/vista.asc | sudo gpg --dearmor -o /etc/apt/keyrings/vista.gpg

# Add the repository with the GPG key verification
echo "deb [signed-by=/etc/apt/keyrings/vista.gpg] https://sebbouez.github.io/Vista-APT/ stable main" | sudo tee /etc/apt/sources.list.d/vista.list

sudo apt update && sudo apt install vista
```
