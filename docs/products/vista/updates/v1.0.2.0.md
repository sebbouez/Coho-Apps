# Vista version 1.0.2.0

Tested on : 
* Linux Mint 22.3 - Cinnamon 64-bit
* Windows 11 26H1 - 64-bit

## Bug Fixes

* Better handling of errors when loading or saving files
* Removed empty commands in **keyboard shortcuts** dialog
* Fixed a bad positioning of the highlight rectangle around shapes
* Fixed a bug that caused pasted shapes to persist even after undoing the operation
* Fixed a weird behavior of snapping helper guides sometimes
* Fixed a bug that allowed to select shapes that belonged to a locked layer
* Fixed a bug that caused the **Opacity** setting of shapes to be reset to 100% when opening a file

## Changes

* Added **New diagram** command in **File** menu
* You can now resize multiple shapes at once using multiple selection (CTRL+click or drag select)
* Changed the rendering of points decorators of selected connections : the first point is now a solid circle and the last one looks like a target
* Added the **Search** feature
* Added tooltips to the default UI toolbar
* Refined some UI elements
