# Vista version 1.0.3.0

Tested on :

* Linux Mint 22.3 - Cinnamon 64-bit
* Windows 11 26H1 - 64-bit

## Bug Fixes

* Fixed a crash in the **About Vista** dialog after opening the **License** file.
* Fixed a crash in the **About Vista** dialog after opening the **Open source references** file.

## Changes

* None.
