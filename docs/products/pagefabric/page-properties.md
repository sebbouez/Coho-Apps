# Page properties

> [!IMPORTANT]
> This product is discontinued.  
> The documentation is available for historical purposes only.


The Page properties panel helps you configure a unique page.

## Styles and layout

The Styles and layout section shows display properties to change the way the page will look.

| Field            | Description                                                                                      |
|------------------|--------------------------------------------------------------------------------------------------|
| Background color | Use it to change the background color of the page, without having to change the whole site theme |
| Text color       | Use it to change the text color of the page, without having to change the whole site theme       |

## Search Engine Optimisation options

This section helps you optimize your position in search engines.

| Field                    | Description                                                                                              |
|--------------------------|----------------------------------------------------------------------------------------------------------|
| Page title               | The title of your page as displayed in search engine results                                             |
| Page description         | The short description displayed in search engine results. Also used for the search itself                |
| Keywords                 | The keywords for search itself                                                                           |
| Edit social network card | Allows you to edit the card that will be shown on social networks when the page if referenced by its URL |
