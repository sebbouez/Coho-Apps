## Working with Parts

|                      |                                                                                                              |
|----------------------|--------------------------------------------------------------------------------------------------------------|
| **Cards**            | Cards are a special way to present content in beautiful containers.                                          |
| **Mosaics**          | Create beautiful mosaics to make your website more modern.                                                   |
| **Images**           | Add images to your pages to make more visual content for your visitors.                                      |
| **Videos**           | Add videos to create more dynamic web pages.                                                                 |
| **Navigation menus** | Add reusable menus to all your pages.                                                                        |
| **Data tables**      | Add data in tables that adapt to all screens.                                                                |
| **Tabs**             | Group data using tabs.                                                                                       |
| **Charts**           | Present your data with beautiful charts.                                                                     |
| **Carousels**        | Add beautiful carousels to add more dynamism to your pages.                                                  |
| **Accordions**       | Organize your content in collapsible groups.                                                                 |
| **Shared parts**     | Create your custom parts once. Use them across multiple pages. Modifications will be reflected on all pages. |
