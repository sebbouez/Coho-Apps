# Layouts and templates

> [!IMPORTANT]
> This product is discontinued.  
> The documentation is available for historical purposes only.


## Layouts

If you want to change the page layout and add menus on the sides:

1. Click the **Layout > Change page layout** button.
2. Select a layout in the **Layouts** dialog.
3. Click the **Apply** button.

## Content margins

If you want to change the content margins, follow these steps:

1. Click the **Layout > Content margins** button.
2. Select the desired margins configuration.

> Headers and footers always span the full width of the page.  
> To center content in a header or footer, use a **Box** and apply a **Box Width** setting.

***

## Page templates

When managing a large website, you don’t want to copy/paste UI elements across many pages — any update would quickly become tedious.  
For that scenario, you should use **page templates**.

> **Note:**  
> When applying a template to a page, the existing layout and margin settings are overridden by those defined in the template.

***

### Create a page template

To create a page template, start from an existing page:

1. Create a new page (it will be deleted at the end of the process).
2. Design the layout you want with header, menus, and footer. Don’t add content — it will belong to child pages.
3. Once satisfied, click **Layout > Convert to template**.
4. In the dialog, set a name for your new template.
5. Your temporary content page will then be deleted.


### Apply a template to an existing page

If you want to apply a template to an existing page:

1. Go to **Layout tab > Change page template**.
2. The page template dialog opens.
3. Select a template and click **Select**.
4. The page will be updated to reflect the chosen template.


### Modify an existing page template

To view and edit existing templates:

1. Go to **Website tab > Page templates**.
2. The page templates dialog opens.
3. Select a template from the list.
4. Click **Edit** to open it in a new editor tab — or **Delete** to remove it.
5. After editing, if changes don’t appear in your pages, click **Layout > Refresh template**.

> **Note:**  
> If you delete a template that’s used by one or more pages, those pages might not display properly when published.  
> Always ensure templates are no longer in use before deletion.
