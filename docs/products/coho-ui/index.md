# Coho.UI

> [!IMPORTANT]
> This product is discontinued.  
> The documentation is available for historical purposes only.

Coho.UI is the WPF open source version of PageFabric UI.  
It contains the shell to create a Windows desktop app using the Sun Valley design plus various other components to create an immersive UI on Windows 10 and 11.

> [!TIP]
> You can download the latest version from NuGet.  
> https://www.nuget.org/packages/Coho.UI/
