# LineStyle Enumeration

| Value (`int`) | Name                             |
|---------------|----------------------------------|
| 0             | Solid line                       |
| 1             | Dashed line                      |
| 2             | Dotted line                      |
| 3             | Line with pattern : Dash Dot     |
| 4             | Line with pattern : Dash Dot Dot |
| 5             | Double solid line                |
