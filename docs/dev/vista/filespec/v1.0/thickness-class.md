# Thickness Class

Represents a thickness.  
A `string` that contains all four sides of a thickness using this format: "left,top,right,bottom".

## Examples

| Value (`string`) | Description                                                          |
|------------------|----------------------------------------------------------------------|
| 10,20,30,40      | 10px on the left, 20px on top, 30px on the right, 40px at the bottom |
| 10,20,10,20      | 10px on the sides, 20px on top and bottom                            |
