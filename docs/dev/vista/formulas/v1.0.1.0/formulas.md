# Formulas


## DATE(format)
This formula returns the current date.

### Arguments
* `string` format : The format of the date to return. If not provided, the default format is used.

### Examples

```text
=DATE()
=DATE("dd/MM/yyyy")
=DATE("dd MM yyyy")
```


## TIME(format)
This formula returns the current time.

### Arguments
* `string` format : The format of the time to return. If not provided, the default format is used.

### Examples

```text
=TIME()
```
