# Coho Apps Trust Center

While no software is absolutely perfect, Coho Apps takes security and privacy seriously.  
This Trust Center provides information on our commitment to these principles and how we ensure the integrity of our products.  

## Code quality and security

All source code is scanned for security vulnerabilities using industry-standard tools and practices, for example, we use:
Qodana from JetBrains and CodeQL from GitHub.

We use few external libraries and services, and select them with care.  
We use Dependabot to keep our dependencies up to date and regularly scan for security vulnerabilities.  

Finally, we regularly review our code using AI to detect security issues.

## Privacy and data protection

None of our apps collects any personal data.  
We don't use any analytics or tracking services.
